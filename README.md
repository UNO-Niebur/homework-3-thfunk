# Homework 3 – Board Game System

## Purpose
This assignment focuses on:
- organizing programs using functions
- reading and writing data from files
- representing and updating program state

Detailed instructions are provided in Canvas.  
This README explains what you need to do **inside this repository**.

---

## Overview

You will create a simple board game system that:
- reads game data from a file
- displays the current state of the game
- allows basic interaction (such as moving a player)

You may work individually or with one partner.

---

## Files You Will Create

### BoardGame.py
This is your main program.

Your program should:
- load game data from a file
- display the game state
- allow basic interaction (for example, moving a player)
- use functions to organize your code

---

### Event / Data File

You must include a file that stores:
- player positions
- events or objects on the board

Example format:

3: Player1

4: Player2

17: Hotel

20: Troll


You may design your own format, but it must be:
- consistent
- readable
- usable by your program

---

### Design Document

You must include a design document that describes:

- what your game looks like
- how the game is played
- how your data is stored
- what functions you are using

This can be:
- a text file
- a shared document (Google Docs, Word, etc.)
- or a simple diagram

---

## Requirements

Your program must:

- Read data from a file  
- Display the game state clearly  
- Allow at least one interaction (move a player, change position, etc.)  
- Use multiple functions to organize the program  

---

## Optional Features (Not Required)

You may choose to:
- create a web interface using Flask  
- add visual elements  
- expand game rules  

These are optional and will not affect your grade if omitted.

---

## What is NOT required

You do NOT need to:
- build a full or polished game  
- implement complex game rules  
- use Flask or web development tools  
- use ChatGPT or external generators  

Focus on structure and functionality.

---

## Submission

To submit Homework 3:

1. Ensure all required files are included:
   - `BoardGame.py`
   - data/event file
   - design document  

2. Commit your work to GitHub  

3. Submit your repository link on Canvas  

---

## Reminder

This assignment is about:
- organizing code  
- using functions  
- working with file data  

It is not about creating a perfect game.
